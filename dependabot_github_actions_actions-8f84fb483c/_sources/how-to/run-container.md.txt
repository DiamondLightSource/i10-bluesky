# Run in a container

Pre-built containers with i10_bluesky and its dependencies already
installed are available on [Github Container Registry](https://ghcr.io/Relm-Arrowny/i10_bluesky).

## Starting the container

To pull the container from github container registry and run:

```
$ docker run ghcr.io/relm-arrowny/i10_bluesky:latest --version
```

To get a released version, use a numbered release instead of `latest`.
